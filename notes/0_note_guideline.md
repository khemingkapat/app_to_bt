# Note Guideline
## Definition
This will be directory for note taking, just in case we need to take some note for this repo, this would be the most appropriate place for it  

make sure your note is brief and clear, utilizing AI assistant but also need to understand it

## Naming Convention
`<yy-mm-dd>_<topic>.md`, for example `26-06-12_eform.md`

> what if we have multiple note on the same day : DO NOT put it in folder, keep it linear and separated by the trailing `<topic>` in the file name.
